# 洛克王国世界精灵蛋查询站

一个基于前端静态部署的查询站。  
输入精灵蛋的 **直径（m）** 与 **重量（kg）**，即可返回可能孵化的候选精灵与概率。

- 在线地址（GitHub Pages）：`https://mfskys.github.io/rocomegg/`

---

## 使用说明

1. 在页面输入精灵蛋尺寸与重量  
2. 点击查询按钮  
3. 查看完全命中、范围命中与近似候选结果
